Diferentes titulos

As Tags 'h1 até h6' são utilizadas para cabeçalhos ou titulos. Cada uma tem um tamanho diferente, para serem utilizadas de diferentes maneira.

Paragrafos

A Tag 'p' é essencial para criar texto, é fundamental para colocar conteúdo na página.

Palavras Destacadas

A tag 'mark' é utilizada para destacar parte do texto dentro de um parágrafo ou outro elemento de texto. Ela é usada para chamar a atenção do leitor para informações importantes

Informações adicionais

A tag 'small' é usada para definir um texto com tamanho de fonte menor do que o texto ao redor. Geralmente, ela é utilizada para fornecer informações adicionais , como notas de rodapé, avisos ou qualquer texto que seja menos importante que o conteúdo principal.

Efeitos de texto

Temos as tags 'i', 'u' e 'strong' 'hr' Elas são usadas em textos para destacar algumas informações como a tag 'i' é usada para colocar texto em itálico. E 'hr' cria uma linha horizontal para separar o conteúdo da página. A 'u' para sublinhar texto. Já a 'strong' é coloca o texto em negrito.

Listas ordenadas e não ordenadas

Para listas ordenadas temos a Tag 'ol' que forma uma lista em que os itens são numerados. Já listas não ordenada usamos 'ul' onde os itens não são numerados. e 'li' é usada dentros de 'ol' ou 'ul' significa um item da lista

Links

Usamos a tag 'a' para criar links Youtube, permitindo que os usuários cliquem em um texto ou imagem para navegar para outra página.

Outras tags

A tag 'font' usada para definir o estilo de fonte de um texto, como cor, tamanho e tipo de letra.

A tag 'del' é usada para indicar texto que foi removido ou deletado do documento. Isso é útil em situações como revisões de documentos ou histórico de alterações.

A tag 'abbr' é usada para definir uma abreviação ou sigla e pode incluir uma explicação completa

A tag 'blockquote

é usada para criar um bloco de texto que é uma citação de outro autor ou fonte.

A tag 'sup' é usada para exibir texto como sobrescrito, o que é comumente usado para exponenciação ou índices em equações.

A tag 'sub' é usada para exibir texto como subscrito, o que é útil para fórmulas químicas, equações matemáticas e outras notações técnicas.
